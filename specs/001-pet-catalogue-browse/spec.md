# Pet Catalogue Browse

## Overview
Build a pet catalogue experience where visitors can browse pets, filter the list, and manage individual pet listings through a clean React interface backed by Spring Boot.

## Scope
- Browse pet cards in a responsive gallery.
- Filter by search text, species, and availability.
- Create, edit, and delete pets through the API.
- Persist pet data in PostgreSQL.

## Out of Scope
- Real payment processing.
- User authentication.
- Multi-role administration.

## Acceptance Criteria
- The gallery loads data from the backend API.
- Filters update the visible results.
- Create, update, and delete operations are reflected immediately in the UI.
- The backend stores data in PostgreSQL, not in memory for normal runtime.
